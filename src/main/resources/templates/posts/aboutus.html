Although there is a proliferation of resources to learn how to code online, nothing can replace the human element of learning alongside others who are on similar learning journeys. We created Social Code so that coders could have a convenient place to be able to come together and easily schedule video chats with others to learn and collaborate.

Social Code is a full-stack application created using the MVC design pattern and is built with Spring Boot, Hibernate, Thymeleaf, Java, JQuery, Javascript, CSS, HTML, and utilizes Integration Testing.
